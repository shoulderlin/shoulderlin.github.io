關於本站

這是一個集合各種實用辦公小工具的網站，包含雙邊文字比對、跨PDF搜尋、QR Code 產生器、密碼產生器、倒數計時器與抽籤工具等，方便您快速完成日常任務。

工具簡介：

  • 雙邊文字比對：用來比對兩段文字的差異，適合文件版本比對。
  • 跨PDF搜尋：用來跨多個PDF檔案，進行關鍵字搜尋，適合快速查閱多個PDF文件。
  • QR Code 產生器：將網址或文字轉為可掃描的 QR Code。
  • 日曆產生器：將產生指定日期區間的日曆，可用來記錄日常。
  • 週曆產生器：將產生指定日期區間的週曆，可用來製作桌曆。
  • 密碼產生器：快速產生強密碼，提升帳號安全。
  • 倒數計時器：簡易計時工具，可設定時間與鬧鈴提醒。
  • 抽籤工具：提供隨機選擇功能，適合分組、抽獎等用途。
